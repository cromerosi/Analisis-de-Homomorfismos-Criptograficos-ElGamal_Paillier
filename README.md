# Analisis-de-Homomorfismos-Criptograficos-ElGamal_Paillier
Simulador en Java que compara el cifrado homomórfico de ElGamal y Paillier en votación electrónica. Evalúa analíticamente cómo sumar votos encriptados dentro de la urna sin romper el anonimato individual, midiendo el rendimiento computacional de ambas estructuras algebraicas sin usar librerías criptográficas externas.
